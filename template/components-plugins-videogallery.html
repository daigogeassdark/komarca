Video Gallery

Huro is integrated with LightgalleryJs, a performant vanilla javascript lightbox library. You can check the plugin documentation on Github. You can also access the javascript code by visiting the assets/js/components.js file. The following example shows you how it works with youtube videos. Works also with Vimeo. 


      //JS CODE
      lightGallery(document.getElementById('videogallery'));
